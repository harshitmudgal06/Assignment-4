# Python Assignment 4

## Overview

This assignment demonstrates the use of Python programming concepts such as:

* Loops
* Functions
* Conditional Statements
* Object-Oriented Programming (OOP)
* Exception Handling
* File Handling

## Programs Included

### 1. Prime Numbers in a List

Prints all prime numbers from a given list using loops.

### 2. Multiplication Table Matrix

Generates a 10 × 10 multiplication table using nested loops.

### 3. Even Count and Odd Sum

Uses a user-defined function to:

* Count even numbers
* Calculate the sum of odd numbers

### 4. Bank Account Class

Implements a `BankAccount` class with:

* Private balance attribute
* Deposit method
* Withdraw method
* Controlled balance access

### 5. Division with Exception Handling

Performs division of two user-input numbers and handles:

* Division by zero
* Invalid input types

### 6. File Processing

Reads numbers from `num.txt` and calculates:

* Total
* Average

Also handles the case when the file does not exist.

## Requirements

* Python 3.x

## How to Run

1. Open a terminal or command prompt.
2. Navigate to the project folder.
3. Run the desired Python file:

```bash
python filename.py
```
